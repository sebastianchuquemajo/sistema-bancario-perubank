# 🏦 PerúBank - Sistema Bancario y Gestión de Divisas en Java

Aplicación de escritorio desarrollada en Java orientada a la simulación de operaciones bancarias de un cajero interactivo, administración segura de cuentas y gestión financiera transaccional. El sistema integra lógica de negocio estructurada bajo Programación Orientada a Objetos (POO), interfaces gráficas intuitivas (GUI) y persistencia de datos relacional mediante MySQL.

## 🚀 Funcionalidades Principales
- **Control y Autenticación:** Registro y validación estricta de credenciales y cuentas de usuario para un acceso seguro.
- **Operaciones de Cajero:** Simulación transaccional para consulta de saldos, depósitos y retiros sincronizados en tiempo real con la base de datos.
- **Motor de Conversión de Divisas:** Algoritmo integrado para el cambio automatizado de Soles (PEN) a Dólares (USD) con control de flujo y alta precisión en el cálculo financiero.
- **Interfaz Gráfica (GUI):** Formularios interactivos desarrollados nativamente para escritorio con Java Swing.

## 🛠️ Stack Tecnológico y Arquitectura
- **Lenguaje Principal:** Java (POO)
- **Base de Datos & Persistencia:** MySQL (Script relacional e instrucciones provistas en `banco.sql`)
- **Conectividad:** Integración mediante JDBC (driver referenciado en `/lib`)
- **Entorno de Desarrollo:** NetBeans IDE / Estructura compatible con Ant (`build.xml`)
